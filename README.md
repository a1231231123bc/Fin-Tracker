# Fin Tracker (Telegram Bot + WebApp)

Теперь проект состоит из двух частей:
- Telegram бот для ввода и аналитики в чате
- Telegram Mini App (WebApp) с дашбордом на реальных данных группы

## Что работает

- Ввод расходов обычным текстом: `450 кофе`, `55 такси`, `продукты 2300`
- Команды: `/today`, `/month`, `/last`, `/undo`, `/settings`, `/app`
- Автокатегоризация по смысловому словарю
- Ежедневные напоминания (настраиваются в `/settings`)
- WebApp: команда `/app` в группе открывает дашборд этой группы

## Запуск

```bash
cp .env.example .env
# заполнить BOT_TOKEN при необходимости

docker compose up --build -d
```

Сервисы:
- bot: `fin-tracker-bot`
- web: `fin-tracker-web` на `http://localhost:8089`

## Важно для Telegram WebApp

Чтобы Mini App открывался внутри Telegram на мобильном/десктопе, `WEBAPP_URL` должен быть публичным HTTPS URL.
Для локальной разработки `http://localhost:8089` подходит только для браузерной проверки.

## Пример

1. В группе: `/setup AED`
2. Пиши расходы обычными сообщениями
3. Открой дашборд: `/app`
